Python for Beginners

Python Basics Study Plan (2-Week Plan)

This plan is designed for absolute beginners to get comfortable with Python fundamentals. It includes daily topics, exercises, and small projects.

📌 Week 1: Getting Started with Python

Day 1: Introduction to Python & Setup
  • 🔹 Install Python and an IDE (VS Code, PyCharm, or Jupyter Notebook)
  • 🔹 Learn how to run Python scripts (print("Hello, World!"))
  • 🔹 Understand basic syntax, comments, and indentation

📝 Exercise: Print your name and a short introduction using print()

Day 2: Variables & Data Types
  • 🔹 Variables (x = 10, name = "Alice")
  • 🔹 Basic data types: int, float, str, bool
  • 🔹 Type conversion (int("5") → 5)

📝 ExerciseExercise: Create a program that stores your name, age, and favorite number, then prints them.

Day 3: Basic Operators
  • 🔹 Arithmetic (+, -, *, /, //, %, **)
  • 🔹 Comparison (==, !=, >, <,>=, <=)
  • 🔹 Logical (and, or, not)

📝 ExerciseExercise: Write a simple calculator that takes two numbers and prints their sum, difference, product, and quotient.

Day 4: Strings & String Manipulation
  • 🔹 String operations (len(), upper(), lower(), replace(), split())
  • 🔹 String formatting (f"Hello, {name}!")

📝 Exercise: Ask the user for their full name and print it in uppercase and reversed order.

Day 5: Lists & Tuples
  • 🔹 Lists (my_list = [1, 2, 3]) and operations (append(), remove(), sort())
  • 🔹 Tuples (my_tuple = (1, 2, 3)) and immutability
  • 🔹 Indexing and slicing (list[0], list[-1], list[1:3])

📝 Exercise: Create a list of your favorite foods and modify it.

Day 6: Conditional Statements
  • 🔹 if, elif, else
  • 🔹 Nested conditions
  • 🔹 Logical conditions with operators

📝 Exercise: Write a program that asks for a user's age and tells them if they are a child, teenager, or adult.

Day 7: Mini Project - Basic Chatbot
  • ✅ Use input(), if-else, and string operations
  • ✅ Let the chatbot greet the user and respond to basic phrases like "hello" or "how are you?"

📌 Week 2: Control Structures & Functions

Day 8: Loops (For & While)
  • 🔹 for loops (iterating over a list or range)
  • 🔹 while loops (looping until a condition is met)
  • 🔹 break and continue

📝 Exercise: Print numbers 1-10 using both for and while loops.

Day 9: Functions & Parameters
  • 🔹 Defining functions (def my_function())
  • 🔹 Parameters and return values
  • 🔹 Default arguments

📝 Exercise: Create a function that takes a name as input and prints a personalized greeting.

Day 10: Dictionaries & Sets
  • 🔹 Dictionaries ({"name": "Alice", "age": 25})
  • 🔹 Adding, modifying, and deleting key-value pairs
  • 🔹 Sets and their properties

📝 Exercise: Create a dictionary storing a few of your favorite movies and their release years.

Day 11: Working with Files
  • 🔹 Reading & writing files (open("file.txt", "w"))
  • 🔹 Appending data to a file
  • 🔹 Reading line by line

📝 Exercise: Write a program that takes user input and saves it to a text file.

Day 12: Error Handling
  • 🔹 Using try-except blocks to catch errors
  • 🔹 Handling user input errors
  • 🔹 Custom error messages

📝 Exercise: Write a program that asks for a number and handles errors if the user enters text.

Day 13: Final Mini Project - To-Do List App
  • ✅ Create a simple to-do list program that allows users to:
  • 🔹 Add tasks
  • 🔹 View tasks
  • 🔹 Remove completed tasks
  • 🔹 Save tasks to a file
Day 14: Review & Next Steps
  • ✅ Review all topics by solving small coding challenges
  • ✅ Resources for further learning:
    • 🔹 FreeCodeCamp Python Course
    • 🔹 Python Crash Course (Book)
    • 🔹 Python Exercises on LeetCode/HackerRank

🔥 Bonus Challenge (Optional): Build a Number Guessing Game

  • 🔹 The program generates a random number
  • 🔹 The user guesses the number
  • 🔹 The program provides hints ("too high" or "too low")
  • 🔹 The game ends when the user guesses correctly
